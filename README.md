# Patient Management

## Getting Started

![Alt Homepage](/src/assets/interface.png)

Install all dependencies

```bash
npm i
# or
yarn
# or
pnpm i
```

Run development server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Run json-server

```bash
npx json-server -p 3001 --watch db.json
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result
